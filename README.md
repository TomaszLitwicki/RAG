# RAG

AI-powered Personal Landing Page with RAG (Retrieval-Augmented Generation), allowing visitors to explore my portfolio, projects, skills, and professional experience through natural-language queries.

## Project overview

This project is an MVP of a **RAG-based Personal Landing Page** built with **Python**, **Flask**, **HTML**, and **CSS**.

The goal is to let visitors interact with my portfolio through an AI-powered interface that retrieves relevant information from a prepared knowledge base instead of relying only on a static website.

## MVP scope

The MVP focuses on:

- serving a Personal Landing Page with Flask
- integrating a simple RAG workflow
- storing and using prepared **chunks** of portfolio content
- using a **manifest** file to organize and describe data sources
- returning context-aware answers based on my professional background, projects, skills, and learning path

## Tech stack

- **Python**
- **Flask**
- **HTML**
- **CSS**
- **RAG (Retrieval-Augmented Generation)**
- structured content in **chunks**
- **manifest** for source management

## Project structure

The project includes:

- **backend** in Flask
- **frontend** in HTML/CSS
- a knowledge base built from **chunks**
- a **manifest** file describing the indexed content
- logic for retrieval and response generation
